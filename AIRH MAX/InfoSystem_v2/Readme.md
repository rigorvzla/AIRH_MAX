# 📊 LibreHardwareMonitorWrapper - Monitoreo de Hardware para AV-AIRH

## 📦 Descripción General

**LibreHardwareMonitorWrapper** es un wrapper en .NET para **LibreHardwareMonitor**, diseñado específicamente para integrar **monitoreo de hardware en tiempo real** con AV-AIRH MAX. Permite al asistente acceder a información de temperatura, voltaje, carga de CPU/GPU, velocidades de ventilador y más, todo desde una API sencilla y unificada.

### 🎯 ¿Para quién es?

- ✅ AV-AIRH MAX y su módulo de monitoreo del sistema
- ✅ Aplicaciones que necesitan **estadísticas de hardware en tiempo real**
- ✅ Asistentes que responden preguntas sobre el estado del PC
- ✅ Desarrolladores que integran monitoreo en .NET

---

## ✨ Características Principales

| Característica | Estado | Descripción |
| :--- | :--- | :--- |
| **Monitor CPU** | ✅ | Temperatura, carga, frecuencia, voltaje |
| **Monitor GPU** | ✅ | Temperatura, carga, frecuencia, memoria |
| **Monitor RAM** | ✅ | Uso total, disponible, porcentaje |
| **Monitor Disco** | ✅ | Temperatura, uso, lectura/escritura |
| **Monitor Ventiladores** | ✅ | Velocidad RPM |
| **Monitor Batería** | ✅ | Nivel, tiempo restante (laptops) |
| **Monitor Red** | ✅ | Velocidad de descarga/subida |
| **Eventos en tiempo real** | ✅ | Alertas por temperatura alta |
| **Formato amigable** | ✅ | Respuestas listas para TTS |
| **100% offline** | ✅ | Sin dependencia de APIs externas |
