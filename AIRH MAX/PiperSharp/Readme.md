# 🔊 PipeSharp - Síntesis de Voz Nativa para AV-AIRH

## 📦 Descripción General

**PipeSharp** es una librería liviana de **síntesis de voz (Text-to-Speech)** para .NET, que actúa como puente nativo con el motor TTS de Windows. Diseñada específicamente para **AV-AIRH MAX**, permite convertir texto a voz de forma **rápida, offline y con baja latencia**, utilizando las voces instaladas en el sistema.

### 🎯 ¿Para quién es?

- ✅ Aplicaciones .NET que necesitan salida de voz
- ✅ Proyectos que requieren TTS **100% offline**
- ✅ AV-AIRH MAX y sus componentes de respuesta por voz
- ✅ Asistentes virtuales, lectores de pantalla, automatizaciones

---

## ✨ Características Principales

| Característica | Estado | Descripción |
| :--- | :--- | :--- |
| **TTS nativo Windows** | ✅ | Usa SpeechSynthesizer integrado |
| **100% offline** | ✅ | Sin dependencia de APIs externas |
| **Baja latencia** | ✅ | Respuesta en <100ms |
| **Múltiples voces** | ✅ | Selecciona voz por nombre o índice |
| **Control de velocidad** | ✅ | Velocidad de -10 a +10 |
| **Control de volumen** | ✅ | Volumen de 0 a 100 |
| **Salida a archivo** | ✅ | Guarda audio como WAV |
| **Cancelación** | ✅ | Detiene síntesis en cualquier momento |
| **Asíncrono** | ✅ | No bloquea la UI |
| **Eventos** | ✅ | Inicio, fin, progreso, error |
