# ⚡ MAX - Módulo Complementario para AV-AIRH

## 📦 Descripción General

**MAX** es un módulo ejecutable complementario diseñado para extender las capacidades de **AV-AIRH MAX**. Actúa como un potenciador del asistente, añadiendo funcionalidades avanzadas que se integran de forma seamless con el ecosistema principal.

### 🎯 ¿Para quién es?

- ✅ Usuarios de AV-AIRH MAX que desean funcionalidades extendidas
- ✅ Equipos que requieren capacidades adicionales del asistente
- ✅ Instalaciones que buscan la máxima experiencia con AIRH
