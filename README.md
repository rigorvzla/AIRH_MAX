# 🎙️ AV-AIRH MAX - Asistente Virtual por Voz con IA Local

[![Versión](https://img.shields.io/badge/version-7.2.0-blue.svg)](https://github.com/rigorvzla/AIRH_MAX/releases)
[![Licencia](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![Plataforma](https://img.shields.io/badge/platform-Windows-0078d7.svg)]()

**Transforma tus palabras en acción.**  
AV-AIRH MAX es un asistente de inteligencia artificial activado por voz, diseñado para mantener conversaciones naturales, entender el contexto y ejecutar tareas en tu computadora. **Olvida los clics: trabaja hablando.**

Actualmente cuenta con la capacidad de usar **OpenCode** como cerebro y podrás **maxificar su eficiencia con MCP** (Model Context Protocol), llevando la productividad por voz al siguiente nivel.

---

## 📖 Historia del Proyecto

Este proyecto **nació en 2016** como una idea personal de **RigorVzla**, y desde entonces ha sido desarrollado de forma **100% independiente y sin fines de lucro**. Durante todos estos años, el desarrollador ha dedicado gran parte de su tiempo, noches y fines de semana a construir, mejorar y mantener este asistente.

## 🚀 Instalación Rápida

1. **Descarga** la versión adecuada desde la sección [Releases](https://github.com/rigorvzla/AIRH_MAX/releases)
2. **Ejecuta** el instalador para tu sistema operativo (Windows)
3. **Sigue** el asistente de configuración inicial (calibración de micrófono y voz)

---

## 🎮 Primeros Pasos

Una vez instalado, activa a AIRH con tu palabra de activación (ej: "Oye AIRH") y prueba estos comandos:

- *"Abre el navegador y busca el clima."*
- *"¿Cuál es la hora?"*
- *"Crea un recordatorio que diga: comprar leche."*
- *"Reproduce música relajante."*

📖 Para dominar **todos los comandos y funciones avanzadas**, consulta el [Manual de Usuario](https://github.com/rigorvzla/AIRH_MAX/blob/main/AIRH%20MAX/AIRH%20MAX/Manual.pdf).
